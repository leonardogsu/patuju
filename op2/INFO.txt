DESCRIPCIÓN:
Sistema completo de instalación automatizada de WordPress multi-sitio usando 
Docker, Nginx, PHP-FPM, MySQL y Let's Encrypt SSL.

CARACTERÍSTICAS PRINCIPALES:
✓ Instalación 100% automatizada
✓ Multi-sitio (múltiples dominios WordPress)
✓ Docker Compose containerizado
✓ Nginx + PHP 8.2 + MySQL 8.0
✓ SSL automático con Let's Encrypt
✓ phpMyAdmin (opcional)
✓ Servidor FTP (opcional)
✓ Sistema de backup automático
✓ Firewall configurado
✓ Scripts de gestión incluidos

CONTENIDO DEL PAQUETE:
├── auto-install.sh          → Instalador principal (EJECUTAR ESTE)
├── README.md                → Documentación completa
├── QUICKSTART.md            → Guía rápida de inicio
├── LICENSE                  → Licencia MIT
├── .gitignore              → Configuración de Git
└── scripts/                 → Scripts auxiliares
    ├── install.sh           → Instalador base
    ├── generate-config.sh   → Generador de configuraciones
    ├── setup.sh             → Setup de WordPress
    ├── setup-ssl.sh         → Configuración SSL
    ├── backup.sh            → Sistema de backup
    ├── manage.sh            → Gestor interactivo
    └── uninstall.sh         → Desinstalador

REQUISITOS:
- Ubuntu 24.04 LTS
- 8GB RAM (mínimo 4GB)
- 20GB espacio libre
- Acceso root/sudo
- IP pública

INSTALACIÓN RÁPIDA:
1. chmod +x auto-install.sh
2. sudo ./auto-install.sh
3. Seguir las instrucciones en pantalla

DESPUÉS DE LA INSTALACIÓN:
- Los sitios estarán en: /opt/wordpress-multisite/
- Gestión: cd /opt/wordpress-multisite && sudo ./scripts/manage.sh
- Backup: cd /opt/wordpress-multisite && sudo ./scripts/backup.sh
